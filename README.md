## Supported accounts
### Token
- Mint
- Account

### Orca Whirlpool
- Whirlpool
- WhirlpoolsConfig
- FeeTier
- Position
- TickArray
- PositionBundle

### Orca Legacy pool (Aquafarm)
- TokenSwap
- GlobalFarm
- UserFarm
